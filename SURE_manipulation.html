ArcGIS API for JavaScript - Mesh Manipulation Sample App

The ArcGIS API for JavaScript includes the ability to modify I3S integrated mesh scene layers to mask, clip, and flatten regions of the mesh. The modification happens client-side in the browser and does not modify the underlying data.

To explore this capability:

  1. Zoom into an area of interest on the mesh using the mouse or the navigation buttons
  2. Select a type of modifcation to create by choosing one of the options
  3. Click the "Create a modification" button to activate the modification tool
  4. Click multiple times on the mesh to drawing a polygon that will capture the extent of the modification
  5. Double-click to complete the polygon
  6. Click the "Apply" button to display the modification

Modifications can also be created on the desktop in ArcGIS Pro and then published along with the scene layer for access from web clients.

The integrated mesh scene layer in this example was generated by nFrames SURE (now SURE for ArcGIS) and is used courtesy of Aerowest GmbH!
